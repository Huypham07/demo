\begin{verbatim}

|   |-- pipeline.yml          # Tham số pipeline
|   `-- train.yml             # Siêu tham số huấn luyện
|-- data/
|   |-- extracted/            # Dữ liệu văn bản thô
|   `-- labels/               # Dữ liệu gán nhãn (topic, action)
|-- src/
|   |-- pipeline/
|   |   |-- run.py            # CLI chạy phân tích
|   |   |-- pipeline.py       # ESGWashingPipeline class
|   |   |-- document_loader.py    # Tải PDF / TXT, OCR qua Docling
|   |   |-- evidence_detector.py  # Phát hiện loại bằng chứng
|   |   |-- evidence_linker.py    # Liên kết bằng chứng (TF-IDF + SBERT)
|   |   |-- nli_verifier.py       # Xác minh NLI (mDeBERTa)
|   |   `-- ewri.py               # Tính WRS và EWRI
|   `-- training/
|       |-- train_model.py        # Huấn luyện phân loại
|       |-- neuro_symbolic.py     # Semantic loss
|       |-- tune_hyperparams.py   # Tối ưu siêu tham số (Optuna)
|       |-- corpus/
|       |   `-- build_corpus.py   # Tiền xử lý dữ liệu
|       `-- labeling/
|           |-- grounded_rules.py     # Bộ quy tắc ký hiệu

\end{verbatim}

\section*{Hướng dẫn chạy}

\subsection*{Cài đặt môi trường}

Yêu cầu Python 3.10 trở lên. Cài đặt dependencies:

\begin{verbatim}
pip install -r requirements.txt
\end{verbatim}

\subsection*{1. Phân tích một báo cáo thường niên}

\begin{verbatim}
python -m src.pipeline.run \
  --input  path/to/annual_report.pdf \
  --output outputs/demo \
  --bank   "TEN_NGAN_HANG" \
  --year   2024
\end{verbatim}

Kết quả xuất ra thư mục \texttt{outputs/demo/}: tệp \texttt{report.html} (báo cáo HTML gồm điểm EWRI, phân rã thành phần, top-10 câu rủi ro cao nhất), \texttt{enriched.parquet} (dữ liệu mức câu đầy đủ).

\subsection*{2. Huấn luyện mô hình phân loại}

\begin{verbatim}
python -m src.training.train_model --task topic
python -m src.training.train_model --task action
\end{verbatim}

Model được lưu vào \texttt{outputs/models/\{topic,action\}\_classifier/final/}.
